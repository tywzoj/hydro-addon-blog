## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/tywzoj/hydro-addon-blog.git
    cd hydro-addon-blog
    ```

2. Apply the addon to your Hydro instance:

    ```bash
    hydrooj addon add /path/to/hydro-addon-blog
    pm2 restart hydrooj
    ```
